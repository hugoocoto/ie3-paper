# Typst IEEE Paper Template

*Unofficial* Template. Non-affiliated with IEEE.

## Usage

1. Copy `lib.typ` to your project root.
2. Import: `#import "lib.typ": ieee`
3. Configure: `#show: ieee.with(...)` (see `./lib.typ` for all the options)
4. Write your paper

By default it uses **English**, `./template/main.typ` uses the **Spanish
translation**.

## Original template

This template expands
[charged-ieee](https://github.com/alerque/typst-universe/tree/main/packages/preview/charged-ieee/0.1.4) 
with multi-language support, summary and better code blocks.
